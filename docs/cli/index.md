# CLI Reference

The Prism CLI includes commands for managing prisme.dev subdomains.

## Commands Overview

| Command | Description |
|---------|-------------|
| `prism auth login` | Authenticate with prisme.dev |
| `prism auth logout` | Remove stored credentials |
| `prism auth status` | Check authentication status |
| `prism subdomain list` | List your subdomains |
| `prism subdomain claim` | Claim a new subdomain |
| `prism subdomain activate` | Activate with IP address |
| `prism subdomain status` | Check DNS propagation |
| `prism subdomain release` | Release a subdomain |
